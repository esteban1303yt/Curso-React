REPOSITORIO GITHUB

Iniciar repositorio local:
"git init"

Cambiar el nombre master a main:
"git branch -M main"

Eliminar credenciales de otros usuarios:
"Win + S"
Escribir: adminstrador de credenciales
Seleccionar credenciales de windows y luego buscar las cuentas de github asociadas y elimina las credenciales.
Luego de eliminar las credenciales

Configurar usuario local de Git:
git config --local user.name "Tu Nombre"
git config --local user.email "tu@email.com"

Ahora agregar todos los archivos del proyecto:
"git add ."

Realizar un commit descriptivo de la versión a subir:
git commit -m "Primera subida"

Enlazar en repositoriode la nube con el local
git remote add origin https://github.com/esteban1303yt/Curso-React.git
git push -u origin main

Luego saldrá que debe iniciar sesión con el navegador:
Inicie sesión con sus credenciales del GitHub.
Listo actualiza el repositorio de la nube

Comprobar el estado del proyecto:
git status

(Si aparece en algo en rojo signfica que el archivo es nuevo o se ha modificado).

Comandos para actualizar el repositorio con la versión local:
git status
git add .
git commit -m "Nombre de la versión o descripción del commit"
git push

Crear ramas:
git checkout -b nombre_rama
git switch -c nombre_rama
git push origin nombre_rama

Crear proyecto react:
npm create vite@latest

javascript + SWC
